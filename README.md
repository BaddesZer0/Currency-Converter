Program Hakkında:

Bu program kodlandığı zaman dilimindeki kur değerleri için dönüşüm yapmaktadır,işlem yapılabilen döviz çeşitleri aşağıda belirtilmiştir:

Programda mevcut,dönüşüm yapılabilen döviz seçenekleri:

a)USD

b)EUR

c)GBP

Nasıl Çalıştırılır?:

1)Program çalıştırıldığında size ilk olarak yapılacak dönüşümün yönünü sormaktadır,TL -> DÖVİZ için "DÖVİZ ALIMI",DÖVİZ -> TL için "DÖVİZ SATIŞI" girmeniz gerekmektedir.

2)İlk adımda girdiyi girerken tüm harflerin büyük olması gerekmektedir,aksi halde hata mesajı alırsınız.

3)Yapılmasını istediğiniz dönüşümün hangi döviz için yapılacağını seçiniz,işlem yapılabilen dövizler harici bir döviz seçilirse sistem tarafından uyarılırsınız.

4)Dönüşümün hesaplanması için mevcut olan TL/DÖVİZ miktarını giriniz.

5)Dönüşüm için girilen miktar pozitif sayı olmalıdır,tersi durumunda sistem sizi uyarır.

6)Seçtiğiniz işlem,istediğiniz kur için dönüştürülür ve ekranda görüntülenir.
